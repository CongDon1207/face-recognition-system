# Thiết lập môi trường (Setup)

## 1. Tạo Virtual Environment
```bash
python -m venv venv
```

## 2. Kích hoạt môi trường

**Windows (CMD/PowerShell):**
```bash
.\venv\Scripts\activate
```

**Git Bash:**
```bash
source venv/Scripts/activate
```

## 3. Cài đặt thư viện
```bash
pip install -r requirements.txt
```
